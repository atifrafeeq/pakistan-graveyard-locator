---

## 🚀 Features

- 🗺️ **Geospatial Mapping** — Every grave pinned to GPS coordinates using Folium
- 📊 **Live Analytics** — 6 real-time metrics from raw SQL
- 📱 **QR Code Generation** — Auto-generated scannable QR per grave
- 🔐 **Security** — bcrypt password hashing + session guards
- 🔍 **Public Search** — Search by name, city, or graveyard
- 🏗️ **Clean Architecture** — 4-layer system
